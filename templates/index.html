Overview

Designed for practical early-stage molecular prioritization

MolScore-PK is intended for descriptor-based molecular profiling rather than docking score prediction, binding affinity estimation, or experimentally measured pharmacokinetic output. The platform is built to support structured screening, rule-based interpretation, medicinal chemistry alerting, and comparison of small molecules in a consistent report-oriented workflow.

Capabilities

Core platform functions

Key features available in the current server version.

🧪

Input Modes

Single SMILES, batch SMILES, local JSME drawing, and CSV upload using a smiles column.

📊

Descriptor Calculation

Molecular weight, LogP, HBD, HBA, TPSA, rotatable bonds, heavy atoms, ring content, aromaticity, and fraction Csp3.

🧬

Drug-Likeness Rules

Lipinski, Veber, Ghose, Rule of 3, and lead-like criteria for practical medicinal chemistry filtering.

🚨

Medicinal Chemistry Alerts

PAINS alerts, Brenk alerts, and synthetic accessibility support where available in the active environment.

Scientific Scope

How current outputs should be interpreted

Current outputs are descriptor-based and rule-based assessments. They should not be interpreted as experimentally validated pharmacokinetic endpoints or target-specific activity predictions.

Included Assessments

  • BBB assessment
  • Absorption assessment
  • Permeability assessment
  • Solubility risk
  • Flexibility risk
  • Polarity, size, and hydrophobicity classes

Output Format

  • Summary table
  • Detailed molecule report cards
  • RDKit 2D structure image
  • Radar-style visualization
  • CSV export for downstream analysis
