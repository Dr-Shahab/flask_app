Overview

MolScore-PK is a web-based molecular profiling platform designed for early-stage compound evaluation using descriptor-based analysis and rule-based interpretation. The system computes physicochemical properties, evaluates drug-likeness criteria, identifies medicinal chemistry alerts, and generates a composite prioritization score to support comparative analysis of small molecules.

The platform accepts molecular structures in SMILES format and provides structured outputs including descriptor summaries, classification results, and visualization-ready molecular reports. The results are intended for screening support rather than experimental pharmacokinetic prediction or target-specific activity modeling.

Usage

Step 1. Select a Method to Enter Molecules

The platform supports multiple input methods to accommodate both single-molecule evaluation and small batch screening:

  • Single SMILES: Enter one SMILES string for rapid profiling.
  • Batch SMILES: Enter multiple SMILES (one per line).
  • Draw Molecule: Use the integrated JSME editor to sketch a structure.
  • Upload File: Upload a CSV file containing a column named smiles (maximum 50 molecules).

Step 2. Submit for Profiling

After selecting the input method, submit the molecules using the analysis button. The server processes each molecule individually and generates descriptor-based outputs.

Step 3. Output Structure

The prediction results are presented in two main formats:

  • Summary Table: Provides a compact overview of all molecules, including descriptors, rule-based evaluations, and ranking scores.
  • Molecule Report Cards: Detailed sections for each molecule including structure visualization, descriptor values, radar profile, and interpretation.

Output Description

Calculated Molecular Descriptors

The platform computes key physicochemical properties including molecular weight (MW), LogP, topological polar surface area (TPSA), hydrogen bond donors (HBD), hydrogen bond acceptors (HBA), rotatable bonds, aromatic ring count, fraction Csp3, molar refractivity, and heteroatom count.

Drug-Likeness Evaluation

Multiple rule-based filters are applied to assess drug-likeness:

  • Lipinski Rule of Five
  • Veber Rule
  • Ghose Filter
  • Rule of 3
  • Lead-like criteria

Property-Based Assessment

Descriptor-driven heuristics are used to classify molecular properties:

  • Blood–brain barrier (BBB) tendency
  • Oral absorption potential
  • Permeability classification
  • Solubility risk
  • Flexibility and polarity class

Medicinal Chemistry Alerts

Structural alerts are identified using curated filters:

  • PAINS: potential assay interference motifs
  • Brenk: undesirable structural fragments
  • SA Score: synthetic accessibility estimation

Descriptor-Based Ranking

Each molecule is assigned a normalized composite score derived from selected physicochemical descriptors. This score reflects the balance between size, lipophilicity, polarity, and flexibility.

The ranking score is further categorized into qualitative classes:

  • Excellent
  • Good
  • Moderate
  • Poor

Interpretation

The results generated by MolScore-PK are based on calculated descriptors and rule-based evaluation. They are intended for:

  • Early-stage compound screening
  • Medicinal chemistry comparison
  • Property-based prioritization
  • Pre-filtering before QSAR or docking workflows

Note:

  • The outputs are not experimental pharmacokinetic data.
  • The ranking is not target-specific.
  • Invalid SMILES cannot be evaluated.
  • Final decisions require expert validation.

Technical Summary

  • Backend: Flask-based Python web application
  • Cheminformatics: RDKit for molecular processing and descriptor calculation
  • Editor: JSME molecular drawing interface
  • Future Work: QSAR modeling, machine learning scoring, docking integration
