Overview

Designed for practical early-stage molecular prioritization

MolScore-PK enables rapid molecular evaluation through descriptor-based profiling, rule-based medicinal chemistry interpretation, and PK-oriented heuristic assessment in a clean scientific interface suitable for comparative screening and structured reporting.

🧪

Flexible Molecular Input

Submit a single SMILES, batch SMILES, upload a CSV file, or draw molecules interactively.

📊

Descriptor Profiling

Calculate molecular weight, LogP, TPSA, HBD, HBA, rotatable bonds, ring content, and related features.

🧬

Drug-Likeness Filtering

Apply Lipinski, Veber, Ghose, Rule of 3, and lead-like evaluation for medicinal chemistry screening.

🚨

Property and Alert Assessment

Review BBB tendency, absorption, permeability, flexibility, polarity, PAINS, Brenk, and synthetic accessibility.

Scientific Scope

What the platform currently delivers

The current server focuses on descriptor-based and rule-based molecular profiling. It is intended for early-stage compound triage, medicinal chemistry comparison, and structured interpretation rather than experimental pharmacokinetic prediction or target-specific bioactivity estimation.

Recommended Use

  • Early-stage compound screening
  • Medicinal chemistry comparison
  • Pre-filtering before QSAR or docking
  • Small-batch prioritization
  • Descriptor-aware report generation

Current Output Style

  • Descriptor summary table
  • Molecule-level scientific report cards
  • 2D structure rendering
  • Radar-style visualization
  • CSV export for downstream use
